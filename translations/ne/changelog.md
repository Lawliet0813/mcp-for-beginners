# परिवर्तनको सूची: MCP प्रारम्भिक पाठ्यक्रम

यो दस्तावेज़ले मोडेल कन्टेक्स्ट प्रोटोकल (MCP) प्रारम्भिक पाठ्यक्रममा गरिएका सबै महत्वपूर्ण परिवर्तनहरूको रेकर्ड राख्छ। परिवर्तनहरू उल्टो कालानुक्रमिक क्रममा (नयाँ परिवर्तनहरू पहिले) दस्तावेज गरिएको छ।

## सेप्टेम्बर १५, २०२५

### उन्नत विषय विस्तार - कस्टम ट्रान्सपोर्ट्स र कन्टेक्स्ट इन्जिनियरिङ

#### MCP कस्टम ट्रान्सपोर्ट्स (05-AdvancedTopics/mcp-transport/) - नयाँ उन्नत कार्यान्वयन मार्गदर्शिका
- **README.md**: कस्टम MCP ट्रान्सपोर्ट मेकानिजमहरूको पूर्ण कार्यान्वयन मार्गदर्शिका
  - **Azure Event Grid Transport**: सर्वरलेस इभेन्ट-ड्राइभन ट्रान्सपोर्ट कार्यान्वयन
    - C#, TypeScript, र Python उदाहरणहरू Azure Functions एकीकरणसहित
    - स्केलेबल MCP समाधानहरूको लागि इभेन्ट-ड्राइभन आर्किटेक्चर ढाँचाहरू
    - वेबहुक रिसिभरहरू र पुश-आधारित सन्देश ह्यान्डलिङ
  - **Azure Event Hubs Transport**: उच्च-थ्रुपुट स्ट्रिमिङ ट्रान्सपोर्ट कार्यान्वयन
    - कम विलम्बता परिदृश्यहरूको लागि वास्तविक-समय स्ट्रिमिङ क्षमता
    - पार्टिशनिङ रणनीतिहरू र चेकप्वाइन्ट व्यवस्थापन
    - सन्देश ब्याचिङ र प्रदर्शन अनुकूलन
  - **उद्यम एकीकरण ढाँचाहरू**: उत्पादन-तयार आर्किटेक्चरल उदाहरणहरू
    - धेरै Azure Functions मार्फत वितरित MCP प्रशोधन
    - धेरै ट्रान्सपोर्ट प्रकारहरू संयोजन गर्ने हाइब्रिड ट्रान्सपोर्ट आर्किटेक्चरहरू
    - सन्देश स्थायित्व, विश्वसनीयता, र त्रुटि ह्यान्डलिङ रणनीतिहरू
  - **सुरक्षा र निगरानी**: Azure Key Vault एकीकरण र अवलोकनीयता ढाँचाहरू
    - प्रबन्धित पहिचान प्रमाणीकरण र न्यूनतम विशेषाधिकार पहुँच
    - Application Insights टेलिमेट्री र प्रदर्शन निगरानी
    - सर्किट ब्रेकरहरू र गल्ती सहनशीलता ढाँचाहरू
  - **परीक्षण फ्रेमवर्कहरू**: कस्टम ट्रान्सपोर्टहरूको लागि व्यापक परीक्षण रणनीतिहरू
    - टेस्ट डबल्स र मोकिङ फ्रेमवर्कहरूसहित युनिट परीक्षण
    - Azure Test Containersसहित एकीकरण परीक्षण
    - प्रदर्शन र लोड परीक्षण विचारहरू

#### कन्टेक्स्ट इन्जिनियरिङ (05-AdvancedTopics/mcp-contextengineering/) - उदयमान AI अनुशासन
- **README.md**: कन्टेक्स्ट इन्जिनियरिङलाई उदयमान क्षेत्रको रूपमा व्यापक अन्वेषण
  - **मुख्य सिद्धान्तहरू**: पूर्ण कन्टेक्स्ट साझेदारी, कार्य निर्णय सचेतता, र कन्टेक्स्ट विन्डो व्यवस्थापन
  - **MCP प्रोटोकल समन्वय**: MCP डिजाइनले कन्टेक्स्ट इन्जिनियरिङ चुनौतीहरू कसरी सम्बोधन गर्छ
    - कन्टेक्स्ट विन्डो सीमाहरू र प्रगतिशील लोडिङ रणनीतिहरू
    - प्रासंगिकता निर्धारण र गतिशील कन्टेक्स्ट पुनःप्राप्ति
    - बहु-मोडल कन्टेक्स्ट ह्यान्डलिङ र सुरक्षा विचारहरू
  - **कार्यान्वयन दृष्टिकोणहरू**: सिंगल-थ्रेडेड बनाम बहु-एजेन्ट आर्किटेक्चरहरू
    - कन्टेक्स्ट चङ्किङ र प्राथमिकता प्रविधिहरू
    - प्रगतिशील कन्टेक्स्ट लोडिङ र कम्प्रेसन रणनीतिहरू
    - स्तरित कन्टेक्स्ट दृष्टिकोणहरू र पुनःप्राप्ति अनुकूलन
  - **मापन फ्रेमवर्क**: कन्टेक्स्ट प्रभावकारिता मूल्याङ्कनका लागि उदयमान मेट्रिक्स
    - इनपुट दक्षता, प्रदर्शन, गुणस्तर, र प्रयोगकर्ता अनुभव विचारहरू
    - कन्टेक्स्ट अनुकूलनका लागि प्रायोगिक दृष्टिकोणहरू
    - असफलता विश्लेषण र सुधार पद्धतिहरू

## अगस्ट १८, २०२५

### दस्तावेज व्यापक अद्यावधिक - MCP 2025-06-18 मानक

#### MCP सुरक्षा उत्तम अभ्यासहरू (02-Security/) - पूर्ण आधुनिकीकरण
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: MCP विशिष्टता 2025-06-18 सँग सुसंगत पूर्ण पुनर्लेखन
  - **अनिवार्य आवश्यकताहरू**: आधिकारिक विशिष्टताबाट स्पष्ट MUST/MUST NOT आवश्यकताहरू थपियो
  - **१२ कोर सुरक्षा अभ्यासहरू**: १५-आइटम सूचीबाट व्यापक सुरक्षा डोमेनहरूमा पुनःसंरचना
    - टोकन सुरक्षा र बाह्य पहिचान प्रदायक एकीकरणसहित प्रमाणीकरण
    - सत्र व्यवस्थापन र ट्रान्सपोर्ट सुरक्षा क्रिप्टोग्राफिक आवश्यकताहरू
    - AI-विशिष्ट खतरा सुरक्षा Microsoft Prompt Shields एकीकरणसहित
    - पहुँच नियन्त्रण र न्यूनतम विशेषाधिकारको सिद्धान्तसहित अनुमति
    - सामग्री सुरक्षा र निगरानी Azure Content Safety एकीकरणसहित
    - आपूर्ति श्रृंखला सुरक्षा व्यापक घटक प्रमाणीकरणसहित
    - OAuth सुरक्षा र Confused Deputy रोकथाम PKCE कार्यान्वयनसहित
    - घटना प्रतिक्रिया र पुनःप्राप्ति स्वचालित क्षमताहरू
    - अनुपालन र शासन नियामक समन्वयसहित
    - उन्नत सुरक्षा नियन्त्रणहरू शून्य विश्वास आर्किटेक्चरसहित
    - Microsoft सुरक्षा पारिस्थितिकी तन्त्र एकीकरण व्यापक समाधानहरूसहित
    - निरन्तर सुरक्षा विकास अनुकूलन अभ्यासहरूसहित
  - **Microsoft सुरक्षा समाधानहरू**: Prompt Shields, Azure Content Safety, Entra ID, र GitHub Advanced Security को लागि उन्नत एकीकरण मार्गदर्शन
  - **कार्यान्वयन स्रोतहरू**: आधिकारिक MCP दस्तावेज, Microsoft सुरक्षा समाधानहरू, सुरक्षा मानकहरू, र कार्यान्वयन मार्गदर्शिकाहरूद्वारा वर्गीकृत व्यापक स्रोत लिंकहरू

#### उन्नत सुरक्षा नियन्त्रणहरू (02-Security/) - उद्यम कार्यान्वयन
- **MCP-SECURITY-CONTROLS-2025.md**: उद्यम-ग्रेड सुरक्षा फ्रेमवर्कसहित पूर्ण पुनर्लेखन
  - **९ व्यापक सुरक्षा डोमेनहरू**: आधारभूत नियन्त्रणहरूबाट विस्तारित विस्तृत उद्यम फ्रेमवर्कमा
    - उन्नत प्रमाणीकरण र अनुमति Microsoft Entra ID एकीकरणसहित
    - टोकन सुरक्षा र Anti-Passthrough नियन्त्रण व्यापक मान्यतासहित
    - सत्र सुरक्षा नियन्त्रण अपहरण रोकथामसहित
    - AI-विशिष्ट सुरक्षा नियन्त्रणहरू Prompt Injection र Tool Poisoning रोकथामसहित
    - Confused Deputy आक्रमण रोकथाम OAuth प्रोक्सी सुरक्षासहित
    - टूल कार्यान्वयन सुरक्षा स्यान्डबक्सिङ र अलगावसहित
    - आपूर्ति श्रृंखला सुरक्षा नियन्त्रण निर्भरता प्रमाणीकरणसहित
    - निगरानी र पत्ता लगाउने नियन्त्रण SIEM एकीकरणसहित
    - घटना प्रतिक्रिया र पुनःप्राप्ति स्वचालित क्षमताहरू
  - **कार्यान्वयन उदाहरणहरू**: विस्तृत YAML कन्फिगरेसन ब्लकहरू र कोड उदाहरणहरू थपियो
  - **Microsoft समाधान एकीकरण**: Azure सुरक्षा सेवाहरू, GitHub Advanced Security, र उद्यम पहिचान व्यवस्थापनको व्यापक कवरेज

#### उन्नत विषय सुरक्षा (05-AdvancedTopics/mcp-security/) - उत्पादन-तयार कार्यान्वयन
- **README.md**: उद्यम सुरक्षा कार्यान्वयनको लागि पूर्ण पुनर्लेखन
  - **वर्तमान विशिष्टता समन्वय**: MCP विशिष्टता 2025-06-18 सँग अद्यावधिक अनिवार्य सुरक्षा आवश्यकताहरू
  - **उन्नत प्रमाणीकरण**: Microsoft Entra ID एकीकरण विस्तृत .NET र Java Spring Security उदाहरणहरूसहित
  - **AI सुरक्षा एकीकरण**: Microsoft Prompt Shields र Azure Content Safety कार्यान्वयन विस्तृत Python उदाहरणहरूसहित
  - **उन्नत खतरा न्यूनीकरण**: व्यापक कार्यान्वयन उदाहरणहरू
    - Confused Deputy आक्रमण रोकथाम PKCE र प्रयोगकर्ता सहमति मान्यतासहित
    - टोकन Passthrough रोकथाम दर्शक मान्यतासहित र सुरक्षित टोकन व्यवस्थापन
    - सत्र अपहरण रोकथाम क्रिप्टोग्राफिक बाइन्डिङ र व्यवहार विश्लेषणसहित
  - **उद्यम सुरक्षा एकीकरण**: Azure Application Insights निगरानी, खतरा पत्ता लगाउने पाइपलाइनहरू, र आपूर्ति श्रृंखला सुरक्षा
  - **कार्यान्वयन चेकलिस्ट**: स्पष्ट अनिवार्य बनाम सिफारिस गरिएको सुरक्षा नियन्त्रणहरू Microsoft सुरक्षा पारिस्थितिकी तन्त्र लाभहरूसहित

  - क्लाइन्ट प्रिमिटिभहरू: स्याम्पलिङ (LLM कम्प्लिशनहरू), एलिसिटेसन (प्रयोगकर्ता इनपुट), लगिङ (डिबगिङ/निगरानी)
  - वर्तमान डिस्कभरी (`*/list`), पुनःप्राप्ति (`*/get`), र कार्यान्वयन (`*/call`) विधि ढाँचाहरूको साथ अद्यावधिक गरियो
- **प्रोटोकल आर्किटेक्चर**: दुई-स्तर आर्किटेक्चर मोडेल प्रस्तुत गरियो
  - डाटा तह: JSON-RPC 2.0 आधार जीवनचक्र व्यवस्थापन र प्रिमिटिभहरूसहित
  - ट्रान्सपोर्ट तह: STDIO (स्थानीय) र Streamable HTTP SSE (टाढाको) ट्रान्सपोर्ट मेकानिजमहरू
- **सुरक्षा फ्रेमवर्क**: स्पष्ट प्रयोगकर्ता सहमति, डाटा गोपनीयता सुरक्षा, टूल कार्यान्वयन सुरक्षा, र ट्रान्सपोर्ट तह सुरक्षासहित व्यापक सुरक्षा सिद्धान्तहरू
- **सञ्चार ढाँचाहरू**: प्रारम्भिककरण, डिस्कभरी, कार्यान्वयन, र सूचना प्रवाह देखाउन अद्यावधिक प्रोटोकल सन्देशहरू
- **कोड उदाहरणहरू**: वर्तमान MCP SDK ढाँचाहरू प्रतिबिम्बित गर्न बहु-भाषा उदाहरणहरू (.NET, Java, Python, JavaScript) ताजा गरियो

- **मानक समन्वय**: MCP विशिष्टता 2025-06-18 सुरक्षा आवश्यकताहरूसँग पूर्ण समन्वय
