# שינויי גרסה: תכנית לימודים MCP למתחילים

#### מושגי ליבה (01-CoreConcepts/) - מודרניזציה מלאה
- **עדכון גרסת פרוטוקול**: עדכון להתייחסות למפרט MCP הנוכחי 2025-06-18 עם גרסה מבוססת תאריך (פורמט YYYY-MM-DD)
- **שיפור ארכיטקטורה**: תיאורים משופרים של מארחים, לקוחות ושרתים המשקפים דפוסי ארכיטקטורה MCP נוכחיים
  - מארחים מוגדרים כעת בבירור כאפליקציות AI המתאמות חיבורי לקוחות MCP מרובים
  - לקוחות מתוארים כמתאמי פרוטוקול השומרים על יחסי אחד-על-אחד עם שרתים
  - שרתים משופרים עם תרחישי פריסה מקומיים מול מרוחקים
- **מבנה פרימיטיבי**: שכתוב מלא של פרימיטיבים לשרתים ולקוחות
  - פרימיטיבים לשרתים: משאבים (מקורות נתונים), פקודות (תבניות), כלים (פונקציות ניתנות לביצוע) עם הסברים ודוגמאות מפורטות
  - פרימיטיבים ללקוחות: דגימה (השלמות LLM), הפקה (קלט משתמש), רישום (ניפוי שגיאות/ניטור)
  - מעודכן עם דפוסי גילוי (`*/list`), שליפה (`*/get`) וביצוע (`*/call`) נוכחיים
- **ארכיטקטורת פרוטוקול**: הוצג מודל ארכיטקטורה דו-שכבתי
  - שכבת נתונים: יסוד JSON-RPC 2.0 עם ניהול מחזור חיים ופרימיטיבים
  - שכבת תעבורה: STDIO (מקומי) ו-HTTP ניתן לזרימה עם SSE (מרוחק)
- **מסגרת אבטחה**: עקרונות אבטחה מקיפים כולל הסכמה מפורשת של משתמש, הגנת פרטיות נתונים, בטיחות ביצוע כלים ואבטחת שכבת תעבורה
- **דפוסי תקשורת**: עדכון הודעות פרוטוקול להצגת זרימות אתחול, גילוי, ביצוע והתראות
- **דוגמאות קוד**: רענון דוגמאות רב-שפתיות (.NET, Java, Python, JavaScript) המשקפות דפוסי SDK MCP נוכחיים
