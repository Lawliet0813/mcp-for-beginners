# পরিবর্তন লগ: MCP ফর বিগিনারস কারিকুলাম

এই ডকুমেন্টটি মডেল কনটেক্সট প্রোটোকল (MCP) ফর বিগিনারস কারিকুলামের সকল গুরুত্বপূর্ণ পরিবর্তনের রেকর্ড হিসেবে কাজ করে। পরিবর্তনগুলো উল্টো ক্রমানুসারে (সর্বশেষ পরিবর্তন প্রথমে) নথিভুক্ত করা হয়েছে।

## ১৫ সেপ্টেম্বর, ২০২৫

### উন্নত বিষয়বস্তু সম্প্রসারণ - কাস্টম ট্রান্সপোর্টস এবং কনটেক্সট ইঞ্জিনিয়ারিং

#### MCP কাস্টম ট্রান্সপোর্টস (05-AdvancedTopics/mcp-transport/) - নতুন উন্নত বাস্তবায়ন গাইড
- **README.md**: কাস্টম MCP ট্রান্সপোর্ট মেকানিজমের সম্পূর্ণ বাস্তবায়ন গাইড
  - **Azure Event Grid Transport**: সার্ভারলেস ইভেন্ট-ড্রিভেন ট্রান্সপোর্ট বাস্তবায়নের বিস্তারিত
    - C#, TypeScript, এবং Python উদাহরণ সহ Azure Functions ইন্টিগ্রেশন
    - স্কেলযোগ্য MCP সমাধানের জন্য ইভেন্ট-ড্রিভেন আর্কিটেকচার প্যাটার্ন
    - Webhook রিসিভার এবং পুশ-ভিত্তিক মেসেজ হ্যান্ডলিং
  - **Azure Event Hubs Transport**: উচ্চ-থ্রুপুট স্ট্রিমিং ট্রান্সপোর্ট বাস্তবায়ন
    - কম লেটেন্সি পরিস্থিতির জন্য রিয়েল-টাইম স্ট্রিমিং সক্ষমতা
    - পার্টিশনিং কৌশল এবং চেকপয়েন্ট ম্যানেজমেন্ট
    - মেসেজ ব্যাচিং এবং পারফরম্যান্স অপ্টিমাইজেশন
  - **এন্টারপ্রাইজ ইন্টিগ্রেশন প্যাটার্ন**: প্রোডাকশন-রেডি আর্কিটেকচারাল উদাহরণ
    - একাধিক Azure Functions-এ বিতরণকৃত MCP প্রসেসিং
    - একাধিক ট্রান্সপোর্ট টাইপের সংমিশ্রণে হাইব্রিড ট্রান্সপোর্ট আর্কিটেকচার
    - মেসেজ টেকসইতা, নির্ভরযোগ্যতা, এবং ত্রুটি হ্যান্ডলিং কৌশল
  - **নিরাপত্তা এবং পর্যবেক্ষণ**: Azure Key Vault ইন্টিগ্রেশন এবং পর্যবেক্ষণ প্যাটার্ন
    - ম্যানেজড আইডেন্টিটি অথেনটিকেশন এবং লিস্ট প্রিভিলেজ অ্যাক্সেস
    - Application Insights টেলিমেট্রি এবং পারফরম্যান্স পর্যবেক্ষণ
    - সার্কিট ব্রেকার এবং ফল্ট টলারেন্স প্যাটার্ন
  - **টেস্টিং ফ্রেমওয়ার্ক**: কাস্টম ট্রান্সপোর্টের জন্য ব্যাপক টেস্টিং কৌশল
    - টেস্ট ডাবল এবং মকিং ফ্রেমওয়ার্ক সহ ইউনিট টেস্টিং
    - Azure Test Containers সহ ইন্টিগ্রেশন টেস্টিং
    - পারফরম্যান্স এবং লোড টেস্টিং বিবেচনা

#### কনটেক্সট ইঞ্জিনিয়ারিং (05-AdvancedTopics/mcp-contextengineering/) - উদীয়মান AI শাখা
- **README.md**: কনটেক্সট ইঞ্জিনিয়ারিংয়ের একটি বিস্তৃত অনুসন্ধান
  - **মূলনীতি**: সম্পূর্ণ কনটেক্সট শেয়ারিং, অ্যাকশন ডিসিশন সচেতনতা, এবং কনটেক্সট উইন্ডো ম্যানেজমেন্ট
  - **MCP প্রোটোকল সামঞ্জস্য**: কিভাবে MCP ডিজাইন কনটেক্সট ইঞ্জিনিয়ারিং চ্যালেঞ্জ মোকাবিলা করে
    - কনটেক্সট উইন্ডো সীমাবদ্ধতা এবং প্রগ্রেসিভ লোডিং কৌশল
    - প্রাসঙ্গিকতা নির্ধারণ এবং ডায়নামিক কনটেক্সট রিট্রিভাল
    - মাল্টি-মোডাল কনটেক্সট হ্যান্ডলিং এবং নিরাপত্তা বিবেচনা
  - **বাস্তবায়ন পদ্ধতি**: সিঙ্গেল-থ্রেডেড বনাম মাল্টি-এজেন্ট আর্কিটেকচার
    - কনটেক্সট চাঙ্কিং এবং অগ্রাধিকার কৌশল
    - প্রগ্রেসিভ কনটেক্সট লোডিং এবং কম্প্রেশন কৌশল
    - স্তরযুক্ত কনটেক্সট পদ্ধতি এবং রিট্রিভাল অপ্টিমাইজেশন
  - **পরিমাপ কাঠামো**: কনটেক্সট কার্যকারিতা মূল্যায়নের উদীয়মান মেট্রিক
    - ইনপুট দক্ষতা, পারফরম্যান্স, গুণমান, এবং ব্যবহারকারীর অভিজ্ঞতা বিবেচনা
    - কনটেক্সট অপ্টিমাইজেশনের পরীক্ষামূলক পদ্ধতি
    - ব্যর্থতা বিশ্লেষণ এবং উন্নয়ন পদ্ধতি
