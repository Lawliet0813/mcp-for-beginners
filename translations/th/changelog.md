# บันทึกการเปลี่ยนแปลง: หลักสูตร MCP สำหรับผู้เริ่มต้น

เอกสารนี้เป็นบันทึกการเปลี่ยนแปลงสำคัญทั้งหมดที่เกิดขึ้นในหลักสูตร Model Context Protocol (MCP) สำหรับผู้เริ่มต้น การเปลี่ยนแปลงจะถูกบันทึกเรียงลำดับจากใหม่ไปเก่า

## 15 กันยายน 2025

### ขยายหัวข้อขั้นสูง - การขนส่งแบบกำหนดเอง & วิศวกรรมบริบท

#### MCP Custom Transports (05-AdvancedTopics/mcp-transport/) - คู่มือการใช้งานขั้นสูงใหม่
- **README.md**: คู่มือการใช้งานสำหรับการขนส่ง MCP แบบกำหนดเอง
  - **Azure Event Grid Transport**: การใช้งานแบบเซิร์ฟเวอร์เลสที่ขับเคลื่อนด้วยเหตุการณ์
    - ตัวอย่างใน C#, TypeScript และ Python พร้อมการผสาน Azure Functions
    - รูปแบบสถาปัตยกรรมที่ขับเคลื่อนด้วยเหตุการณ์สำหรับโซลูชัน MCP ที่ปรับขนาดได้
    - ตัวรับ Webhook และการจัดการข้อความแบบผลักดัน
  - **Azure Event Hubs Transport**: การใช้งานการขนส่งแบบสตรีมมิ่งที่มีความเร็วสูง
    - ความสามารถในการสตรีมแบบเรียลไทม์สำหรับสถานการณ์ที่มีความหน่วงต่ำ
    - กลยุทธ์การแบ่งพาร์ติชันและการจัดการจุดตรวจสอบ
    - การจัดกลุ่มข้อความและการปรับประสิทธิภาพ
  - **Enterprise Integration Patterns**: ตัวอย่างสถาปัตยกรรมที่พร้อมใช้งานในระดับการผลิต
    - การประมวลผล MCP แบบกระจายผ่าน Azure Functions หลายตัว
    - สถาปัตยกรรมการขนส่งแบบไฮบริดที่รวมประเภทการขนส่งหลายแบบ
    - กลยุทธ์ความทนทานของข้อความ ความน่าเชื่อถือ และการจัดการข้อผิดพลาด
  - **Security & Monitoring**: การผสาน Azure Key Vault และรูปแบบการสังเกตการณ์
    - การรับรองความถูกต้องด้วย Managed Identity และการเข้าถึงที่มีสิทธิ์น้อยที่สุด
    - การตรวจสอบประสิทธิภาพด้วย Application Insights
    - รูปแบบ Circuit Breaker และความทนทานต่อข้อผิดพลาด
  - **Testing Frameworks**: กลยุทธ์การทดสอบที่ครอบคลุมสำหรับการขนส่งแบบกำหนดเอง
    - การทดสอบหน่วยด้วย test doubles และ mocking frameworks
    - การทดสอบการผสานด้วย Azure Test Containers
    - การพิจารณาการทดสอบประสิทธิภาพและโหลด

#### Context Engineering (05-AdvancedTopics/mcp-contextengineering/) - สาขา AI ที่กำลังเกิดขึ้นใหม่
- **README.md**: การสำรวจวิศวกรรมบริบทอย่างครอบคลุมในฐานะสาขาที่กำลังเกิดขึ้นใหม่
  - **Core Principles**: การแบ่งปันบริบทอย่างสมบูรณ์ การตระหนักถึงการตัดสินใจ และการจัดการหน้าต่างบริบท
  - **MCP Protocol Alignment**: การออกแบบ MCP ที่ตอบสนองความท้าทายของวิศวกรรมบริบท
    - ข้อจำกัดของหน้าต่างบริบทและกลยุทธ์การโหลดแบบก้าวหน้า
    - การกำหนดความเกี่ยวข้องและการดึงบริบทแบบไดนามิก
    - การจัดการบริบทแบบหลายรูปแบบและการพิจารณาด้านความปลอดภัย
  - **Implementation Approaches**: สถาปัตยกรรมแบบเธรดเดียวและหลายตัวแทน
    - เทคนิคการแบ่งส่วนบริบทและการจัดลำดับความสำคัญ
    - การโหลดบริบทแบบก้าวหน้าและกลยุทธ์การบีบอัด
    - การเข้าถึงบริบทแบบชั้นและการปรับปรุงการดึงข้อมูล
  - **Measurement Framework**: เมตริกที่เกิดขึ้นใหม่สำหรับการประเมินประสิทธิภาพของบริบท
    - ประสิทธิภาพการป้อนข้อมูล ประสิทธิภาพ คุณภาพ และการพิจารณาประสบการณ์ผู้ใช้
    - วิธีการทดลองเพื่อการปรับปรุงบริบท
    - การวิเคราะห์ความล้มเหลวและวิธีการปรับปรุง

#### การอัปเดตการนำทางหลักสูตร (README.md)
- **Enhanced Module Structure**: ปรับปรุงตารางหลักสูตรเพื่อรวมหัวข้อขั้นสูงใหม่
  - เพิ่ม Context Engineering (5.14) และ Custom Transport (5.15)
  - รูปแบบและลิงก์การนำทางที่สอดคล้องกันในทุกโมดูล
  - อัปเดตคำอธิบายเพื่อสะท้อนขอบเขตเนื้อหาปัจจุบัน

### การปรับปรุงโครงสร้างไดเรกทอรี
- **Naming Standardization**: เปลี่ยนชื่อ "mcp transport" เป็น "mcp-transport" เพื่อความสอดคล้องกับโฟลเดอร์หัวข้อขั้นสูงอื่น ๆ
- **Content Organization**: โฟลเดอร์ 05-AdvancedTopics ทั้งหมดตอนนี้มีรูปแบบการตั้งชื่อที่สอดคล้องกัน (mcp-[topic])

### การปรับปรุงคุณภาพเอกสาร
- **MCP Specification Alignment**: เนื้อหาใหม่ทั้งหมดอ้างอิง MCP Specification 2025-06-18
- **Multi-Language Examples**: ตัวอย่างโค้ดที่ครอบคลุมใน C#, TypeScript และ Python
- **Enterprise Focus**: รูปแบบที่พร้อมใช้งานในระดับการผลิตและการผสาน Azure cloud
- **Visual Documentation**: แผนภาพ Mermaid สำหรับการแสดงภาพสถาปัตยกรรมและการไหล

## 18 สิงหาคม 2025

### การอัปเดตเอกสารอย่างครอบคลุม - มาตรฐาน MCP 2025-06-18

#### MCP Security Best Practices (02-Security/) - การปรับปรุงใหม่ทั้งหมด
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: การเขียนใหม่ทั้งหมดที่สอดคล้องกับ MCP Specification 2025-06-18
  - **Mandatory Requirements**: เพิ่มข้อกำหนด MUST/MUST NOT จากสเปคอย่างชัดเจนพร้อมตัวบ่งชี้ภาพ
  - **12 Core Security Practices**: ปรับโครงสร้างจากรายการ 15 รายการเป็นโดเมนความปลอดภัยที่ครอบคลุม
    - ความปลอดภัยของโทเค็น & การรับรองความถูกต้องพร้อมการผสานผู้ให้บริการตัวตนภายนอก
    - การจัดการเซสชัน & ความปลอดภัยของการขนส่งพร้อมข้อกำหนดการเข้ารหัส
    - การป้องกันภัยคุกคามเฉพาะ AI พร้อมการผสาน Microsoft Prompt Shields
    - การควบคุมการเข้าถึง & การอนุญาตตามหลักการสิทธิ์น้อยที่สุด
    - ความปลอดภัยของเนื้อหา & การตรวจสอบด้วยการผสาน Azure Content Safety
    - ความปลอดภัยของห่วงโซ่อุปทานพร้อมการตรวจสอบส่วนประกอบที่ครอบคลุม
    - ความปลอดภัย OAuth & การป้องกัน Confused Deputy พร้อมการใช้งาน PKCE
    - การตอบสนองต่อเหตุการณ์ & การกู้คืนพร้อมความสามารถอัตโนมัติ
    - การปฏิบัติตามข้อกำหนด & การกำกับดูแลที่สอดคล้องกับกฎระเบียบ
    - การควบคุมความปลอดภัยขั้นสูงพร้อมสถาปัตยกรรม Zero Trust
    - การผสานระบบความปลอดภัยของ Microsoft อย่างครอบคลุม
    - การพัฒนาความปลอดภัยอย่างต่อเนื่องพร้อมแนวปฏิบัติที่ปรับตัว
  - **Microsoft Security Solutions**: คำแนะนำการผสานที่ปรับปรุงสำหรับ Prompt Shields, Azure Content Safety, Entra ID และ GitHub Advanced Security
  - **Implementation Resources**: ลิงก์ทรัพยากรการใช้งานที่ครอบคลุมจัดหมวดหมู่ตามเอกสาร MCP อย่างเป็นทางการ โซลูชันความปลอดภัยของ Microsoft มาตรฐานความปลอดภัย และคู่มือการใช้งาน

#### Advanced Security Controls (02-Security/) - การใช้งานระดับองค์กร
- **MCP-SECURITY-CONTROLS-2025.md**: การปรับปรุงใหม่ทั้งหมดพร้อมกรอบความปลอดภัยระดับองค์กร
  - **9 Comprehensive Security Domains**: ขยายจากการควบคุมพื้นฐานเป็นกรอบความปลอดภัยระดับองค์กรที่ละเอียด
    - การรับรองความถูกต้อง & การอนุญาตขั้นสูงพร้อมการผสาน Microsoft Entra ID
    - ความปลอดภัยของโทเค็น & การควบคุม Anti-Passthrough พร้อมการตรวจสอบที่ครอบคลุม
    - การควบคุมความปลอดภัยของเซสชันพร้อมการป้องกันการแย่งชิง
    - การควบคุมความปลอดภัยเฉพาะ AI พร้อมการป้องกันการฉีดคำสั่งและการวางยาพิษเครื่องมือ
    - การป้องกันการโจมตี Confused Deputy พร้อมความปลอดภัย OAuth proxy
    - ความปลอดภัยการดำเนินการเครื่องมือพร้อมการแยกและการกักกัน
    - การควบคุมความปลอดภัยของห่วงโซ่อุปทานพร้อมการตรวจสอบการพึ่งพา
    - การควบคุมการตรวจสอบ & การตรวจจับพร้อมการผสาน SIEM
    - การตอบสนองต่อเหตุการณ์ & การกู้คืนพร้อมความสามารถอัตโนมัติ
  - **Implementation Examples**: เพิ่มบล็อกการกำหนดค่า YAML และตัวอย่างโค้ดที่ละเอียด
  - **Microsoft Solutions Integration**: การครอบคลุมบริการความปลอดภัย Azure, GitHub Advanced Security และการจัดการตัวตนระดับองค์กรอย่างครอบคลุม

#### Advanced Topics Security (05-AdvancedTopics/mcp-security/) - การใช้งานที่พร้อมสำหรับการผลิต
- **README.md**: การเขียนใหม่ทั้งหมดสำหรับการใช้งานความปลอดภัยระดับองค์กร
  - **Current Specification Alignment**: อัปเดตให้สอดคล้องกับ MCP Specification 2025-06-18 พร้อมข้อกำหนดความปลอดภัยที่บังคับ
  - **Enhanced Authentication**: การผสาน Microsoft Entra ID พร้อมตัวอย่าง .NET และ Java Spring Security ที่ครอบคลุม
  - **AI Security Integration**: การใช้งาน Microsoft Prompt Shields และ Azure Content Safety พร้อมตัวอย่าง Python ที่ละเอียด
  - **Advanced Threat Mitigation**: ตัวอย่างการใช้งานที่ครอบคลุมสำหรับ
    - การป้องกันการโจมตี Confused Deputy พร้อม PKCE และการตรวจสอบความยินยอมของผู้ใช้
    - การป้องกัน Token Passthrough พร้อมการตรวจสอบผู้รับและการจัดการโทเค็นที่ปลอดภัย
    - การป้องกันการแย่งชิงเซสชันพร้อมการผูกเข้ารหัสและการวิเคราะห์พฤติกรรม
  - **Enterprise Security Integration**: การตรวจสอบ Azure Application Insights, ท่อการตรวจจับภัยคุกคาม และความปลอดภัยของห่วงโซ่อุปทาน
  - **Implementation Checklist**: การควบคุมความปลอดภัยที่บังคับและแนะนำอย่างชัดเจนพร้อมประโยชน์ของระบบความปลอดภัย Microsoft

### การปรับปรุงคุณภาพเอกสาร & การจัดแนวมาตรฐาน

- **Microsoft Security Ecosystem**: คำแนะนำการผสานที่ปรับปรุงในเอกสารความปลอดภัยทั้งหมด
- **Practical Implementation**: เพิ่มตัวอย่างโค้ดที่ละเอียดใน .NET, Java และ Python พร้อมรูปแบบระดับองค์กร
- **Resource Organization**: การจัดหมวดหมู่ทรัพยากรอย่างครอบคลุมของเอกสารอย่างเป็นทางการ มาตรฐานความปลอดภัย และคู่มือการใช้งาน
- **Visual Indicators**: การทำเครื่องหมายข้อกำหนดที่บังคับและแนวปฏิบัติที่แนะนำอย่างชัดเจน

## 16 กรกฎาคม 2025

### การปรับปรุง README และการนำทาง
- ออกแบบการนำทางหลักสูตรใน README.md ใหม่ทั้งหมด
- แทนที่แท็ก `<details>` ด้วยรูปแบบตารางที่เข้าถึงได้มากขึ้น
- สร้างตัวเลือกเลย์เอาต์ทางเลือกในโฟลเดอร์ "alternative_layouts" ใหม่
- เพิ่มตัวอย่างการนำทางแบบการ์ด แบบแท็บ และแบบแอคคอร์เดียน
- อัปเดตส่วนโครงสร้างที่เก็บเพื่อรวมไฟล์ล่าสุดทั้งหมด
- ปรับปรุงส่วน "How to Use This Curriculum" พร้อมคำแนะนำที่ชัดเจน
- อัปเดตลิงก์สเปค MCP ให้ชี้ไปยัง URL ที่ถูกต้อง
- เพิ่มส่วน Context Engineering (5.14) ในโครงสร้างหลักสูตร

### การอัปเดตคู่มือการศึกษา
- ปรับปรุงคู่มือการศึกษาให้สอดคล้องกับโครงสร้างที่เก็บปัจจุบัน
- เพิ่มส่วนใหม่สำหรับ MCP Clients และ Tools และ MCP Servers ยอดนิยม
- อัปเดตแผนที่หลักสูตรแบบภาพให้สะท้อนหัวข้อทั้งหมดอย่างถูกต้อง
- ปรับปรุงคำอธิบายหัวข้อขั้นสูงให้ครอบคลุมพื้นที่เฉพาะทั้งหมด
- อัปเดตส่วนกรณีศึกษาให้สะท้อนตัวอย่างจริง
- เพิ่มบันทึกการเปลี่ยนแปลงที่ครอบคลุมนี้

### การสนับสนุนชุมชน (06-CommunityContributions/)
- เพิ่มข้อมูลรายละเอียดเกี่ยวกับ MCP servers สำหรับการสร้างภาพ
- เพิ่มส่วนที่ครอบคลุมเกี่ยวกับการใช้ Claude ใน VSCode
- เพิ่มคำแนะนำการตั้งค่าและการใช้งาน Cline terminal client
- อัปเดตส่วน MCP client เพื่อรวมตัวเลือก client ยอดนิยมทั้งหมด
- ปรับปรุงตัวอย่างการสนับสนุนด้วยตัวอย่างโค้ดที่แม่นยำมากขึ้น

### หัวข้อขั้นสูง (05-AdvancedTopics/)
- จัดระเบียบโฟลเดอร์หัวข้อเฉพาะทั้งหมดด้วยรูปแบบการตั้งชื่อที่สอดคล้องกัน
- เพิ่มวัสดุและตัวอย่างวิศวกรรมบริบท
- เพิ่มเอกสารการผสาน Foundry agent
- ปรับปรุงเอกสารการผสานความปลอดภัย Entra ID

## 11 มิถุนายน 2025

### การสร้างครั้งแรก
- เผยแพร่เวอร์ชันแรกของหลักสูตร MCP สำหรับผู้เริ่มต้น
- สร้างโครงสร้างพื้นฐานสำหรับ 10 ส่วนหลักทั้งหมด
- ใช้แผนที่หลักสูตรแบบภาพสำหรับการนำทาง
- เพิ่มโครงการตัวอย่างเริ่มต้นในหลายภาษาโปรแกรม

### การเริ่มต้นใช้งาน (03-GettingStarted/)
- สร้างตัวอย่างการใช้งานเซิร์ฟเวอร์ครั้งแรก
- เพิ่มคำแนะนำการพัฒนา client
- รวมคำแนะนำการผสาน LLM client
- เพิ่มเอกสารการผสาน VS Code
- ใช้ตัวอย่างเซิร์ฟเวอร์ Server-Sent Events (SSE)

### แนวคิดหลัก (01-CoreConcepts/)
- เพิ่มคำอธิบายโดยละเอียดเกี่ยวกับสถาปัตยกรรม client-server
- สร้างเอกสารเกี่ยวกับส่วนประกอบโปรโตคอลสำคัญ
- บันทึกรูปแบบการส่งข้อความใน MCP

## 23 พฤษภาคม 2025

### โครงสร้างที่เก็บ
- เริ่มต้นที่เก็บด้วยโครงสร้างโฟลเดอร์พื้นฐาน
- สร้างไฟล์ README สำหรับแต่ละส่วนหลัก
- ตั้งค่าโครงสร้างพื้นฐานการแปล
- เพิ่มสินทรัพย์ภาพและแผนภาพ

### เอกสาร
- สร้าง README.md เริ่มต้นพร้อมภาพรวมหลักสูตร
- เพิ่ม CODE_OF_CONDUCT.md และ SECURITY.md
- ตั้งค่า SUPPORT.md พร้อมคำแนะนำสำหรับการขอความช่วยเหลือ
- สร้างโครงสร้างคู่มือการศึกษาเบื้องต้น

## 15 เมษายน 2025

### การวางแผนและกรอบงาน
- การวางแผนเบื้องต้นสำหรับหลักสูตร MCP สำหรับผู้เริ่มต้น
- กำหนดวัตถุประสงค์การเรียนรู้และกลุ่มเป้าหมาย
- ร่างโครงสร้าง 10 ส่วนของหลักสูตร
- พัฒนากรอบแนวคิดสำหรับตัวอย่างและกรณีศึกษา
- สร้างตัวอย่างต้นแบบเบื้องต้นสำหรับแนวคิดสำคัญ
