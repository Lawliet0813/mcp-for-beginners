# ਚੇਂਜਲੌਗ: ਸ਼ੁਰੂਆਤੀ MCP ਕਰਿਕੁਲਮ

ਇਹ ਦਸਤਾਵੇਜ਼ ਮਾਡਲ ਕੌਂਟੈਕਸਟ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਲਈ ਸ਼ੁਰੂਆਤੀ ਕਰਿਕੁਲਮ ਵਿੱਚ ਕੀਤੇ ਗਏ ਸਾਰੇ ਮਹੱਤਵਪੂਰਨ ਬਦਲਾਅ ਦਾ ਰਿਕਾਰਡ ਹੈ। ਬਦਲਾਅ ਨੂੰ ਉਲਟ ਕ੍ਰਮ ਵਿੱਚ ਦਰਜ ਕੀਤਾ ਗਿਆ ਹੈ (ਨਵੇਂ ਬਦਲਾਅ ਪਹਿਲਾਂ).

    - **ਲੈਬਜ਼ 07-09: ਉੱਨਤ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ** - ਸੈਮੈਂਟਿਕ ਸਰਚ ਇੰਟੀਗ੍ਰੇਸ਼ਨ, ਟੈਸਟਿੰਗ ਅਤੇ ਡੀਬੱਗਿੰਗ, VS ਕੋਡ ਇੰਟੀਗ੍ਰੇਸ਼ਨ
    - **ਲੈਬਜ਼ 10-12: ਪ੍ਰੋਡਕਸ਼ਨ ਅਤੇ ਬਿਹਤਰ ਅਭਿਆਸ** - ਡਿਪਲੌਇਮੈਂਟ ਰਣਨੀਤੀਆਂ, ਮਾਨੀਟਰਿੰਗ ਅਤੇ ਦ੍ਰਿਸ਼ਟਾਵਲਤਾ, ਬਿਹਤਰ ਅਭਿਆਸ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ
  - **ਐਂਟਰਪ੍ਰਾਈਜ਼ ਤਕਨਾਲੋਜੀਆਂ**: FastMCP ਫਰੇਮਵਰਕ, PostgreSQL ਨਾਲ pgvector, Azure OpenAI embeddings, Azure Container Apps, Application Insights

  - **Azure Event Grid Transport**: ਵਿਸਤ੍ਰਿਤ ਸਰਵਰਲੈਸ ਇਵੈਂਟ-ਡ੍ਰਿਵਨ ਟ੍ਰਾਂਸਪੋਰਟ ਅਮਲਵਾਰੀ

    - ਸਕੇਲਬਲ MCP ਹੱਲਾਂ ਲਈ ਇਵੈਂਟ-ਡ੍ਰਿਵਨ ਆਰਕੀਟੈਕਚਰ ਪੈਟਰਨ
    - ਵੈੱਬਹੁੱਕ ਰਿਸੀਵਰ ਅਤੇ ਪੁਸ਼-ਅਧਾਰਿਤ ਸੁਨੇਹਾ ਸੰਭਾਲ

    - ਕਈ ਟ੍ਰਾਂਸਪੋਰਟ ਕਿਸਮਾਂ ਨੂੰ ਜੋੜਨ ਵਾਲੇ ਹਾਈਬ੍ਰਿਡ ਟ੍ਰਾਂਸਪੋਰਟ ਆਰਕੀਟੈਕਚਰ
    - ਸੁਨੇਹਾ ਟਿਕਾਊਪਨ, ਭਰੋਸੇਯੋਗਤਾ, ਅਤੇ ਗਲਤੀ ਸੰਭਾਲ ਰਣਨੀਤੀਆਂ

    - ਸਰਕਟ ਬ੍ਰੇਕਰ ਅਤੇ ਗਲਤੀ ਸਹਿਣਸ਼ੀਲਤਾ ਪੈਟਰਨ

#### ਕੌਂਟੈਕਸਟ ਇੰਜੀਨੀਅਰਿੰਗ (05-AdvancedTopics/mcp-contextengineering/) - ਉਭਰਦਾ AI ਵਿਸ਼ਾ
- **ਦ੍ਰਿਸ਼ਮਾਨ ਸੰਕੇਤਕ**: ਲਾਜ਼ਮੀ ਜ਼ਰੂਰਤਾਂ ਅਤੇ ਸਿਫਾਰਸ਼ੀ ਅਭਿਆਸਾਂ ਦੀ ਸਪਸ਼ਟ ਪਛਾਣ

#### ਮੁੱਖ ਧਾਰਨਾਵਾਂ (01-CoreConcepts/) - ਪੂਰੀ ਤਰ੍ਹਾਂ ਅਧੁਨਿਕੀਕਰਨ
- **ਪ੍ਰੋਟੋਕੋਲ ਵਰਜਨ ਅਪਡੇਟ**: ਮੌਜੂਦਾ MCP Specification 2025-06-18 ਨੂੰ YYYY-MM-DD ਫਾਰਮੈਟ ਦੇ ਤਰੀਕੇ ਨਾਲ ਦਰਸਾਇਆ
- **ਆਰਕੀਟੈਕਚਰ ਸੁਧਾਰ**: Hosts, Clients, ਅਤੇ Servers ਦੀਆਂ ਵਰਣਨਾਵਾਂ ਨੂੰ ਮੌਜੂਦਾ MCP ਆਰਕੀਟੈਕਚਰ ਪੈਟਰਨਾਂ ਦੇ ਅਨੁਸਾਰ ਵਧਾਇਆ
  - Hosts ਹੁਣ ਸਪਸ਼ਟ ਤੌਰ 'ਤੇ AI ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਜੋਂ ਪਰਿਭਾਸ਼ਿਤ ਕੀਤੇ ਗਏ ਹਨ ਜੋ ਕਈ MCP ਕਲਾਇੰਟ ਕਨੈਕਸ਼ਨਾਂ ਦਾ ਸਮਨੁਯੋਜਨ ਕਰਦੇ ਹਨ
  - Clients ਨੂੰ ਪ੍ਰੋਟੋਕੋਲ ਕਨੈਕਟਰ ਵਜੋਂ ਦਰਸਾਇਆ ਗਿਆ ਹੈ ਜੋ ਇੱਕ-ਤੋਂ-ਇੱਕ ਸਰਵਰ ਸੰਬੰਧਾਂ ਨੂੰ ਬਣਾਈ ਰੱਖਦੇ ਹਨ
  - Servers ਨੂੰ ਸਥਾਨਕ ਅਤੇ ਦੂਰ-ਵਰਤਮਾਨ ਤੈਨਾਤੀ ਦ੍ਰਿਸ਼ਾਂ ਨਾਲ ਵਧਾਇਆ ਗਿਆ
- **ਪ੍ਰਿਮਿਟਿਵਾਂ ਦਾ ਪੁਨਰਗਠਨ**: ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵਾਂ ਦੀ ਪੂਰੀ ਤਰ੍ਹਾਂ ਨਵੀਂ ਬਣਤਰ
  - Server Primitives: Resources (ਡਾਟਾ ਸਰੋਤ), Prompts (ਟੈਂਪਲੇਟ), Tools (ਚਲਾਉਣਯੋਗ ਫੰਕਸ਼ਨ) ਨਾਲ ਵਿਸਤ੍ਰਿਤ ਵਿਆਖਿਆ ਅਤੇ ਉਦਾਹਰਨਾਂ
  - Client Primitives: Sampling (LLM completions), Elicitation (ਯੂਜ਼ਰ ਇਨਪੁਟ), Logging (ਡਿਬੱਗਿੰਗ/ਮਾਨੀਟਰਿੰਗ)
  - ਮੌਜੂਦਾ ਖੋਜ (`*/list`), ਪ੍ਰਾਪਤੀ (`*/get`), ਅਤੇ ਚਲਾਉਣ (`*/call`) ਵਿਧੀ ਪੈਟਰਨਾਂ ਨਾਲ ਅਪਡੇਟ ਕੀਤਾ
- **ਪ੍ਰੋਟੋਕੋਲ ਆਰਕੀਟੈਕਚਰ**: ਦੋ-ਪੱਧਰੀ ਆਰਕੀਟੈਕਚਰ ਮਾਡਲ ਪੇਸ਼ ਕੀਤਾ
  - ਡਾਟਾ ਪੱਧਰ: JSON-RPC 2.0 ਆਧਾਰ ਨਾਲ ਜੀਵਨਚੱਕਰ ਪ੍ਰਬੰਧਨ ਅਤੇ ਪ੍ਰਿਮਿਟਿਵਾਂ
  - ਟ੍ਰਾਂਸਪੋਰਟ ਪੱਧਰ: STDIO (ਸਥਾਨਕ) ਅਤੇ Streamable HTTP ਨਾਲ SSE (ਦੂਰ-ਵਰਤਮਾਨ) ਟ੍ਰਾਂਸਪੋਰਟ ਮਕੈਨਿਜ਼ਮ
- **ਸੁਰੱਖਿਆ ਫਰੇਮਵਰਕ**: ਵਿਸਤ੍ਰਿਤ ਸੁਰੱਖਿਆ ਸਿਧਾਂਤ ਜਿਵੇਂ ਕਿ ਸਪਸ਼ਟ ਯੂਜ਼ਰ ਸਹਿਮਤੀ, ਡਾਟਾ ਗੋਪਨੀਯਤਾ ਸੁਰੱਖਿਆ, ਟੂਲ ਚਲਾਉਣ ਦੀ ਸੁਰੱਖਿਆ, ਅਤੇ ਟ੍ਰਾਂਸਪੋਰਟ ਪੱਧਰ ਸੁਰੱਖਿਆ
- **ਸੰਚਾਰ ਪੈਟਰਨ**: ਪ੍ਰੋਟੋਕੋਲ ਸੁਨੇਹਿਆਂ ਨੂੰ ਅਪਡੇਟ ਕੀਤਾ ਜੋ ਸ਼ੁਰੂਆਤ, ਖੋਜ, ਚਲਾਉਣ, ਅਤੇ ਸੂਚਨਾ ਪ੍ਰਵਾਹ ਦਰਸਾਉਂਦੇ ਹਨ
- **ਕੋਡ ਉਦਾਹਰਨਾਂ**: ਮੌਜੂਦਾ MCP SDK ਪੈਟਰਨਾਂ ਨੂੰ ਦਰਸਾਉਣ ਲਈ ਬਹੁ-ਭਾਸ਼ਾ ਉਦਾਹਰਨਾਂ (.NET, Java, Python, JavaScript) ਨੂੰ ਤਾਜ਼ਾ ਕੀਤਾ

#### ਸੁਰੱਖਿਆ (02-Security/) - ਵਿਸਤ੍ਰਿਤ ਸੁਰੱਖਿਆ ਪੁਨਰਗਠਨ  
- **ਮਿਆਰਾਂ ਨਾਲ ਸੰਗਤੀ**: MCP Specification 2025-06-18 ਸੁਰੱਖਿਆ ਜ਼ਰੂਰਤਾਂ ਨਾਲ ਪੂਰੀ ਸੰਗਤੀ

  - ਹਕੀਕਤੀ ਉਦਾਹਰਨਾਂ ਨਾਲ ਵਿਸਤ੍ਰਿਤ ਪ੍ਰੋਮਪਟ ਇੰਜੈਕਸ਼ਨ ਹਮਲੇ ਦੇ ਦ੍ਰਿਸ਼

- **Microsoft AI ਸੁਰੱਖਿਆ ਹੱਲ**: Microsoft ਸੁਰੱਖਿਆ ਪਰਿਸਰ ਦੀ ਵਿਸਤ੍ਰਿਤ ਕਵਰੇਜ
  - AI Prompt Shields ਨਾਲ ਉੱਚ-ਤਕਨੀਕੀ ਪਛਾਣ, ਸਪਾਟਲਾਈਟਿੰਗ, ਅਤੇ ਡਿਲੀਮੀਟਰ ਤਕਨੀਕ
  - Azure Content Safety ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਪੈਟਰਨ
  - GitHub Advanced Security ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ ਲਈ
- **ਉੱਚ-ਤਕਨੀਕੀ ਖਤਰੇ ਨੂੰ ਘਟਾਉਣਾ**: ਵਿਸਤ੍ਰਿਤ ਸੁਰੱਖਿਆ ਨਿਯੰਤਰਣ
  - ਸੈਸ਼ਨ ਹਾਈਜੈਕਿੰਗ MCP-ਵਿਸ਼ੇਸ਼ ਹਮਲੇ ਦੇ ਦ੍ਰਿਸ਼ ਅਤੇ ਕ੍ਰਿਪਟੋਗ੍ਰਾਫਿਕ ਸੈਸ਼ਨ ID ਜ਼ਰੂਰਤਾਂ ਨਾਲ
  - MCP ਪ੍ਰਾਕਸੀ ਦ੍ਰਿਸ਼ਾਂ ਵਿੱਚ ਗੁੰਝਲਦਾਰ ਡਿਪਟੀ ਸਮੱਸਿਆਵਾਂ ਲਈ ਸਪਸ਼ਟ ਸਹਿਮਤੀ ਜ਼ਰੂਰਤਾਂ
