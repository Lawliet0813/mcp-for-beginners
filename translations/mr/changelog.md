# बदलांचा इतिहास: MCP नवशिक्यांसाठी अभ्यासक्रम

हा दस्तऐवज मॉडेल कॉन्टेक्स्ट प्रोटोकॉल (MCP) नवशिक्यांसाठी अभ्यासक्रमात केलेल्या महत्त्वपूर्ण बदलांचा नोंद ठेवतो. बदल उलट कालानुक्रमाने (नवीन बदल प्रथम) दस्तऐवजीकरण केले आहेत.

## १५ सप्टेंबर, २०२५

### प्रगत विषय विस्तार - कस्टम ट्रान्सपोर्ट्स आणि कॉन्टेक्स्ट इंजिनिअरिंग

#### MCP कस्टम ट्रान्सपोर्ट्स (05-AdvancedTopics/mcp-transport/) - नवीन प्रगत अंमलबजावणी मार्गदर्शक
- **README.md**: कस्टम MCP ट्रान्सपोर्ट यंत्रणेसाठी संपूर्ण अंमलबजावणी मार्गदर्शक
  - **Azure Event Grid Transport**: सर्व्हरलेस इव्हेंट-ड्रिव्हन ट्रान्सपोर्ट अंमलबजावणी
    - C#, TypeScript, आणि Python उदाहरणे Azure Functions एकत्रीकरणासह
    - स्केलेबल MCP सोल्यूशन्ससाठी इव्हेंट-ड्रिव्हन आर्किटेक्चर पॅटर्न्स
    - वेबहूक रिसिव्हर्स आणि पुश-आधारित संदेश हाताळणी
  - **Azure Event Hubs Transport**: उच्च-थ्रूपुट स्ट्रीमिंग ट्रान्सपोर्ट अंमलबजावणी
    - कमी विलंबता परिस्थितीसाठी रिअल-टाइम स्ट्रीमिंग क्षमता
    - विभाजन धोरणे आणि चेकपॉइंट व्यवस्थापन
    - संदेश बॅचिंग आणि कार्यक्षमता अनुकूलन
  - **एंटरप्राइझ इंटिग्रेशन पॅटर्न्स**: उत्पादन-तयार आर्किटेक्चरल उदाहरणे
    - अनेक Azure Functions मध्ये वितरित MCP प्रक्रिया
    - अनेक ट्रान्सपोर्ट प्रकार एकत्रित करणारी हायब्रिड ट्रान्सपोर्ट आर्किटेक्चर
    - संदेश टिकाऊपणा, विश्वासार्हता, आणि त्रुटी हाताळणी धोरणे
  - **सुरक्षा आणि निरीक्षण**: Azure Key Vault एकत्रीकरण आणि निरीक्षण पॅटर्न्स
    - व्यवस्थापित ओळख प्रमाणीकरण आणि कमीतकमी विशेषाधिकार प्रवेश
    - Application Insights टेलीमेट्री आणि कार्यक्षमता निरीक्षण
    - सर्किट ब्रेकर्स आणि फॉल्ट टॉलरन्स पॅटर्न्स
  - **चाचणी फ्रेमवर्क्स**: कस्टम ट्रान्सपोर्ट्ससाठी व्यापक चाचणी धोरणे
    - टेस्ट डबल्स आणि मॉकिंग फ्रेमवर्कसह युनिट चाचणी
    - Azure Test Containers सह इंटिग्रेशन चाचणी
    - कार्यक्षमता आणि लोड चाचणी विचार

#### कॉन्टेक्स्ट इंजिनिअरिंग (05-AdvancedTopics/mcp-contextengineering/) - उदयोन्मुख AI शास्त्र
- **README.md**: कॉन्टेक्स्ट इंजिनिअरिंग या उदयोन्मुख क्षेत्राचा व्यापक अन्वेषण
  - **मूलभूत तत्त्वे**: संपूर्ण कॉन्टेक्स्ट शेअरिंग, कृती निर्णय जागरूकता, आणि कॉन्टेक्स्ट विंडो व्यवस्थापन
  - **MCP प्रोटोकॉल संरेखन**: MCP डिझाइन कॉन्टेक्स्ट इंजिनिअरिंग आव्हानांना कसे संबोधित करते
    - कॉन्टेक्स्ट विंडो मर्यादा आणि प्रोग्रेसिव्ह लोडिंग धोरणे
    - प्रासंगिकता निर्धारण आणि डायनॅमिक कॉन्टेक्स्ट पुनर्प्राप्ती
    - मल्टी-मोडल कॉन्टेक्स्ट हाताळणी आणि सुरक्षा विचार
  - **अंमलबजावणी दृष्टिकोन**: सिंगल-थ्रेडेड विरुद्ध मल्टी-एजंट आर्किटेक्चर
    - कॉन्टेक्स्ट चंकींग आणि प्राधान्यक्रम तंत्र
    - प्रोग्रेसिव्ह कॉन्टेक्स्ट लोडिंग आणि संक्षेपण धोरणे
    - स्तरित कॉन्टेक्स्ट दृष्टिकोन आणि पुनर्प्राप्ती अनुकूलन
  - **मापन फ्रेमवर्क**: कॉन्टेक्स्ट प्रभावीतेचे मूल्यांकन करण्यासाठी उदयोन्मुख मेट्रिक्स
    - इनपुट कार्यक्षमता, कार्यक्षमता, गुणवत्ता, आणि वापरकर्ता अनुभव विचार
    - कॉन्टेक्स्ट अनुकूलनासाठी प्रायोगिक दृष्टिकोन
    - अपयश विश्लेषण आणि सुधारणा पद्धती

## १८ ऑगस्ट, २०२५

### दस्तऐवजीकरण व्यापक अद्यतन - MCP 2025-06-18 मानके

#### MCP सुरक्षा सर्वोत्तम पद्धती (02-Security/) - संपूर्ण आधुनिकीकरण
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: MCP तपशील 2025-06-18 शी संरेखित संपूर्ण पुनर्लेखन
  - **अनिवार्य आवश्यकता**: अधिकृत तपशीलातील स्पष्ट MUST/MUST NOT आवश्यकता स्पष्ट दृश्य निर्देशकांसह जोडल्या
  - **१२ मुख्य सुरक्षा पद्धती**: १५-आयटम यादीतून व्यापक सुरक्षा डोमेनमध्ये पुनर्रचना
    - टोकन सुरक्षा आणि बाह्य ओळख प्रदाता एकत्रीकरणासह प्रमाणीकरण
    - सत्र व्यवस्थापन आणि ट्रान्सपोर्ट सुरक्षा क्रिप्टोग्राफिक आवश्यकताांसह
    - AI-विशिष्ट धोका संरक्षण Microsoft Prompt Shields एकत्रीकरणासह
    - प्रवेश नियंत्रण आणि परवानग्या कमीतकमी विशेषाधिकार तत्त्वासह
    - सामग्री सुरक्षा आणि निरीक्षण Azure Content Safety एकत्रीकरणासह
    - पुरवठा साखळी सुरक्षा व्यापक घटक सत्यापनासह
    - OAuth सुरक्षा आणि Confused Deputy प्रतिबंध PKCE अंमलबजावणीसह
    - घटना प्रतिसाद आणि पुनर्प्राप्ती स्वयंचलित क्षमता असलेली
    - अनुपालन आणि प्रशासन नियामक संरेखनासह
    - प्रगत सुरक्षा नियंत्रण शून्य विश्वास आर्किटेक्चरसह
    - Microsoft सुरक्षा इकोसिस्टम एकत्रीकरण व्यापक उपायांसह
    - सतत सुरक्षा उत्क्रांती अनुकूल पद्धतींसह
  - **Microsoft सुरक्षा उपाय**: Prompt Shields, Azure Content Safety, Entra ID, आणि GitHub Advanced Security साठी वाढवलेली एकत्रीकरण मार्गदर्शन
  - **अंमलबजावणी संसाधने**: अधिकृत MCP दस्तऐवजीकरण, Microsoft सुरक्षा उपाय, सुरक्षा मानके, आणि अंमलबजावणी मार्गदर्शकांद्वारे वर्गीकृत व्यापक संसाधन दुवे

#### प्रगत सुरक्षा नियंत्रण (02-Security/) - एंटरप्राइझ अंमलबजावणी
- **MCP-SECURITY-CONTROLS-2025.md**: एंटरप्राइझ-ग्रेड सुरक्षा फ्रेमवर्कसह संपूर्ण पुनर्लेखन
  - **९ व्यापक सुरक्षा डोमेन**: मूलभूत नियंत्रणांपासून विस्तृत एंटरप्राइझ फ्रेमवर्कमध्ये विस्तार
    - प्रगत प्रमाणीकरण आणि अधिकृतता Microsoft Entra ID एकत्रीकरणासह
    - टोकन सुरक्षा आणि अँटी-पासथ्रू नियंत्रण व्यापक सत्यापनासह
    - सत्र सुरक्षा नियंत्रण हायजॅकिंग प्रतिबंधासह
    - AI-विशिष्ट सुरक्षा नियंत्रण प्रॉम्प्ट इंजेक्शन आणि टूल विषबाधा प्रतिबंधासह
    - Confused Deputy हल्ला प्रतिबंध OAuth प्रॉक्सी सुरक्षा
    - टूल अंमलबजावणी सुरक्षा सॅंडबॉक्सिंग आणि वेगळेपणासह
    - पुरवठा साखळी सुरक्षा नियंत्रण अवलंबित्व सत्यापनासह
    - निरीक्षण आणि शोध नियंत्रण SIEM एकत्रीकरणासह
    - घटना प्रतिसाद आणि पुनर्प्राप्ती स्वयंचलित क्षमता असलेली
  - **अंमलबजावणी उदाहरणे**: तपशीलवार YAML कॉन्फिगरेशन ब्लॉक्स आणि कोड उदाहरणे जोडली
  - **Microsoft उपाय एकत्रीकरण**: Azure सुरक्षा सेवा, GitHub Advanced Security, आणि एंटरप्राइझ ओळख व्यवस्थापनाचे व्यापक कव्हरेज

#### प्रगत विषय सुरक्षा (05-AdvancedTopics/mcp-security/) - उत्पादन-तयार अंमलबजावणी
- **README.md**: एंटरप्राइझ सुरक्षा अंमलबजावणीसाठी संपूर्ण पुनर्लेखन
  - **वर्तमान तपशील संरेखन**: MCP तपशील 2025-06-18 शी अद्यतनित अनिवार्य सुरक्षा आवश्यकताांसह
  - **वाढवलेले प्रमाणीकरण**: Microsoft Entra ID एकत्रीकरण .NET आणि Java Spring Security उदाहरणांसह
  - **AI सुरक्षा एकत्रीकरण**: Microsoft Prompt Shields आणि Azure Content Safety अंमलबजावणी तपशीलवार Python उदाहरणांसह
  - **प्रगत धोका कमी करणे**: व्यापक अंमलबजावणी उदाहरणे
    - Confused Deputy हल्ला प्रतिबंध PKCE आणि वापरकर्ता संमती सत्यापनासह
    - टोकन पासथ्रू प्रतिबंध प्रेक्षक सत्यापन आणि सुरक्षित टोकन व्यवस्थापनासह
    - सत्र हायजॅकिंग प्रतिबंध क्रिप्टोग्राफिक बाइंडिंग आणि वर्तन विश्लेषणासह
  - **एंटरप्राइझ सुरक्षा एकत्रीकरण**: Azure Application Insights निरीक्षण, धोका शोध पाइपलाइन्स, आणि पुरवठा साखळी सुरक्षा
  - **अंमलबजावणी चेकलिस्ट**: स्पष्ट अनिवार्य विरुद्ध शिफारस केलेले सुरक्षा नियंत्रण Microsoft सुरक्षा इकोसिस्टम फायदेांसह

  - सर्व्हर प्रिमिटिव्ह्स: संसाधने (डेटा स्रोत), प्रॉम्प्ट्स (टेम्पलेट्स), टूल्स (कार्यकारी फंक्शन्स) तपशीलवार स्पष्टीकरण आणि उदाहरणांसह
  - क्लायंट प्रिमिटिव्ह्स: सॅम्पलिंग (LLM पूर्णता), एलिसिटेशन (वापरकर्ता इनपुट), लॉगिंग (डिबगिंग/निरीक्षण)
  - वर्तमान डिस्कव्हरी (`*/list`), पुनर्प्राप्ती (`*/get`), आणि अंमलबजावणी (`*/call`) पद्धती पॅटर्न्ससह अद्यतनित
- **प्रोटोकॉल आर्किटेक्चर**: दोन-स्तरीय आर्किटेक्चर मॉडेल सादर केले
  - डेटा स्तर: JSON-RPC 2.0 पाया जीवनचक्र व्यवस्थापन आणि प्रिमिटिव्ह्ससह
  - ट्रान्सपोर्ट स्तर: STDIO (स्थानिक) आणि Streamable HTTP SSE (दूरस्थ) ट्रान्सपोर्ट यंत्रणा
- **सुरक्षा फ्रेमवर्क**: स्पष्ट वापरकर्ता संमती, डेटा गोपनीयता संरक्षण, टूल अंमलबजावणी सुरक्षा, आणि ट्रान्सपोर्ट स्तर सुरक्षा यासह व्यापक सुरक्षा तत्त्वे
- **संप्रेषण पॅटर्न्स**: प्रारंभ, डिस्कव्हरी, अंमलबजावणी, आणि सूचना प्रवाह दर्शविण्यासाठी अद्यतनित प्रोटोकॉल संदेश
- **कोड उदाहरणे**: वर्तमान MCP SDK पॅटर्न्स प्रतिबिंबित करण्यासाठी बहु-भाषा उदाहरणे (.NET, Java, Python, JavaScript) ताजी केली

#### सुरक्षा (02-Security/) - व्यापक सुरक्षा पुनर्लेखन  
- **मानक संरेखन**: MCP तपशील 2025-06-18 सुरक्षा आवश्यकताांसह पूर्ण संरेखन
- **प्रमाणीकरण उत्क्रांती**: कस्टम OAuth सर्व्हर्सपासून बाह्य ओळख प्रदाता प्रतिनिधी (Microsoft Entra ID) पर्यंत उत्क्रांती दस्तऐवजीकरण
- **AI-विशिष्ट धोका विश्लेषण**: आधुनिक AI हल्ला वेक्टरचे विस्तारित कव्हरेज
  - वास्तविक उदाहरणांसह तपशीलवार प्रॉम्प्ट इंजेक्शन हल्ला परिस्थिती
  - टूल विषबाधा यंत्रणा आणि "रग पुल" हल्ला पॅटर्न्स
  - कॉन्टेक्स्ट विंडो विषबाधा आणि मॉडेल गोंधळ हल्ले
- **Microsoft AI सुरक्षा उपाय**: Microsoft सुरक्षा इकोसिस्टमचे व्यापक कव्हरेज
  - AI Prompt Shields प्रगत शोध, स्पॉटलाइटिंग, आणि डिलिमिटर तंत्रांसह
  - Azure Content Safety एकत्रीकरण पॅटर्न्स
  - GitHub Advanced Security पुरवठा साखळी संरक्षणासाठी
- **प्रगत धोका कमी करणे**: MCP-विशिष्ट हल्ला परिस्थितींसह सत्र हायजॅकिंग क्रिप्टोग्राफिक सत्र ID आवश्यकता
  - MCP प्रॉक्सी परिस्थितीतील Confused Deputy समस्या स्पष्ट संमती आवश्यकता
  - टोकन पासथ्रू असुरक्षितता अनिवार्य सत्यापन नियंत्रणांसह
- **पुरवठा साखळी सुरक्षा**: फाउंडेशन मॉडेल्स, एम्बेडिंग्स सेवा, कॉन्टेक्स्ट प्रदाते, आणि तृतीय-पक्ष API यासह विस्तारित AI पुरवठा साखळी कव्हरेज
- **फाउंडेशन सुरक्षा**: शून्य विश्वास आर्किटेक्चर आणि Microsoft सुरक्षा इकोसिस्टमसह एंटरप्राइझ सुरक्षा पॅटर्न्ससह वाढवलेले एकत्रीकरण
- **संसाधन संघटन**: प्रकारानुसार व्यापक संसाधन दुवे वर्गीकृत (अधिकृत दस्तऐवजीकरण, मानके, संशोधन, Microsoft उपाय, अंमलबजावणी मार्गदर्शक)
