# سجل التغييرات: منهج MCP للمبتدئين

يعمل هذا المستند كمرجع لجميع التغييرات المهمة التي تم إجراؤها على منهج بروتوكول سياق النموذج (MCP) للمبتدئين. يتم توثيق التغييرات بترتيب زمني عكسي (أحدث التغييرات أولاً).

## 15 سبتمبر 2025

### توسعة المواضيع المتقدمة - النقل المخصص وهندسة السياق

#### النقل المخصص لـ MCP (05-AdvancedTopics/mcp-transport/) - دليل تنفيذ متقدم جديد
- **README.md**: دليل كامل لتنفيذ آليات النقل المخصصة لـ MCP
  - **نقل Azure Event Grid**: تنفيذ شامل للنقل بدون خوادم يعتمد على الأحداث
    - أمثلة بـ C#، TypeScript، وPython مع تكامل وظائف Azure
    - أنماط معمارية تعتمد على الأحداث لحلول MCP قابلة للتوسع
    - استقبال Webhook ومعالجة الرسائل المستندة إلى الدفع
  - **نقل Azure Event Hubs**: تنفيذ نقل عالي الإنتاجية يعتمد على التدفق
    - قدرات تدفق البيانات في الوقت الفعلي لسيناريوهات منخفضة التأخير
    - استراتيجيات التقسيم وإدارة نقاط التحقق
    - تجميع الرسائل وتحسين الأداء
  - **أنماط التكامل المؤسسي**: أمثلة معمارية جاهزة للإنتاج
    - معالجة MCP الموزعة عبر وظائف Azure متعددة
    - معماريات نقل هجينة تجمع بين أنواع نقل متعددة
    - استراتيجيات استدامة الرسائل والموثوقية ومعالجة الأخطاء
  - **الأمان والمراقبة**: تكامل Azure Key Vault وأنماط المراقبة
    - مصادقة الهوية المدارة والوصول بأقل امتيازات
    - مراقبة الأداء باستخدام Application Insights
    - أنماط الحماية من الأعطال
  - **أطر الاختبار**: استراتيجيات اختبار شاملة للنقل المخصص
    - اختبار الوحدة باستخدام أطر التزييف والمحاكاة
    - اختبار التكامل باستخدام Azure Test Containers
    - اعتبارات اختبار الأداء والتحميل

#### هندسة السياق (05-AdvancedTopics/mcp-contextengineering/) - تخصص ناشئ في الذكاء الاصطناعي
- **README.md**: استكشاف شامل لهندسة السياق كتخصص ناشئ
  - **المبادئ الأساسية**: مشاركة السياق بالكامل، الوعي باتخاذ الإجراءات، وإدارة نافذة السياق
  - **التوافق مع بروتوكول MCP**: كيف يعالج تصميم MCP تحديات هندسة السياق
    - قيود نافذة السياق واستراتيجيات التحميل التدريجي
    - تحديد الصلة واسترجاع السياق الديناميكي
    - معالجة السياق متعدد الوسائط واعتبارات الأمان
  - **طرق التنفيذ**: معماريات أحادية الخيط مقابل متعددة الوكلاء
    - تقطيع السياق وتقنيات تحديد الأولويات
    - تحميل السياق التدريجي واستراتيجيات الضغط
    - نهج السياق الطبقي وتحسين الاسترجاع
  - **إطار القياس**: مقاييس ناشئة لتقييم فعالية السياق
    - كفاءة الإدخال، الأداء، الجودة، واعتبارات تجربة المستخدم
    - نهج تجريبية لتحسين السياق
    - تحليل الفشل ومنهجيات التحسين

#### المفاهيم الأساسية (01-CoreConcepts/) - تحديث شامل
- **تحديث إصدار البروتوكول**: تحديث للإشارة إلى مواصفات MCP الحالية 2025-06-18 مع إصدار يعتمد على التاريخ (YYYY-MM-DD)
- **تحسين المعمارية**: تحسين وصف المضيفين، العملاء، والخوادم ليعكس أنماط معمارية MCP الحالية
  - تعريف المضيفين بوضوح كتطبيقات ذكاء اصطناعي تنسق اتصالات العملاء المتعددة لـ MCP
  - وصف العملاء كموصلات بروتوكول تحافظ على علاقات خادم واحد لواحد
  - تحسين الخوادم مع سيناريوهات النشر المحلي مقابل البعيد
- **إعادة هيكلة البدائيات**: إعادة صياغة كاملة لبدائيات الخادم والعميل
  - بدائيات الخادم: الموارد (مصادر البيانات)، المطالبات (القوالب)، الأدوات (وظائف قابلة للتنفيذ) مع شرح وأمثلة مفصلة
  - بدائيات العميل: أخذ العينات (إكمالات LLM)، الاستنباط (مدخلات المستخدم)، التسجيل (التصحيح/المراقبة)
  - تحديث مع أنماط الاكتشاف (`*/list`)، الاسترجاع (`*/get`)، والتنفيذ (`*/call`)
- **معمارية البروتوكول**: تقديم نموذج معماري ذو طبقتين
  - طبقة البيانات: أساس JSON-RPC 2.0 مع إدارة دورة الحياة والبدائيات
  - طبقة النقل: STDIO (محلي) وHTTP قابل للبث مع SSE (بعيد)
- **إطار الأمان**: مبادئ أمان شاملة تشمل موافقة المستخدم الصريحة، حماية خصوصية البيانات، أمان تنفيذ الأدوات، وأمان طبقة النقل
- **أنماط الاتصال**: تحديث رسائل البروتوكول لإظهار التدفق الأولي، الاكتشاف، التنفيذ، والإشعارات
- **أمثلة بالكود**: تحديث أمثلة متعددة اللغات (.NET، Java، Python، JavaScript) لتعكس أنماط SDK الحالية لـ MCP

#### الأمان (02-Security/) - تحديث شامل للأمان  
- **التوافق مع المعايير**: توافق كامل مع متطلبات أمان مواصفات MCP 2025-06-18
- **تطور المصادقة**: توثيق التطور من خوادم OAuth المخصصة إلى تفويض مزود هوية خارجي (Microsoft Entra ID)
- **تحليل تهديدات محددة للذكاء الاصطناعي**: تغطية محسنة لنواقل الهجوم الحديثة للذكاء الاصطناعي
  - سيناريوهات هجوم حقن المطالبات مع أمثلة واقعية
  - آليات تسميم الأدوات وأنماط هجوم "سحب البساط"
  - تسميم نافذة السياق وهجمات تشويش النموذج
- **حلول أمان الذكاء الاصطناعي من Microsoft**: تغطية شاملة لنظام أمان Microsoft
  - دروع المطالبات الذكية مع تقنيات الكشف المتقدمة، تسليط الضوء، والفواصل
  - أنماط تكامل Azure Content Safety
  - GitHub Advanced Security لحماية سلسلة التوريد
- **التخفيف من التهديدات المتقدمة**: ضوابط أمان مفصلة لـ
  - اختطاف الجلسات مع سيناريوهات هجوم محددة لـ MCP ومتطلبات معرف الجلسة المشفر
  - مشاكل Confused Deputy في سيناريوهات وكيل MCP مع متطلبات موافقة صريحة
  - ثغرات تمرير الرموز مع ضوابط تحقق إلزامية
- **أمان سلسلة التوريد**: تغطية موسعة لسلسلة توريد الذكاء الاصطناعي بما في ذلك النماذج الأساسية، خدمات التضمين، مزودي السياق، وواجهات برمجة التطبيقات الخارجية
- **أمان الأساس**: تحسين التكامل مع أنماط الأمان المؤسسية بما في ذلك معمارية الثقة الصفرية ونظام أمان Microsoft
- **تنظيم الموارد**: تصنيف شامل للروابط المرجعية حسب النوع (وثائق رسمية، معايير، أبحاث، حلول Microsoft، أدلة التنفيذ)
