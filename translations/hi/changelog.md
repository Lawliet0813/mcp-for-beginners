# चेंजलॉग: शुरुआती लोगों के लिए MCP पाठ्यक्रम

यह दस्तावेज़ शुरुआती लोगों के लिए मॉडल कॉन्टेक्स्ट प्रोटोकॉल (MCP) पाठ्यक्रम में किए गए सभी महत्वपूर्ण बदलावों का रिकॉर्ड है। बदलावों को उल्टे कालानुक्रमिक क्रम में (नवीनतम बदलाव पहले) दर्ज किया गया है।

## 15 सितंबर, 2025

### उन्नत विषय विस्तार - कस्टम ट्रांसपोर्ट्स और कॉन्टेक्स्ट इंजीनियरिंग

#### MCP कस्टम ट्रांसपोर्ट्स (05-AdvancedTopics/mcp-transport/) - नया उन्नत कार्यान्वयन गाइड
- **README.md**: कस्टम MCP ट्रांसपोर्ट तंत्र के लिए पूर्ण कार्यान्वयन गाइड
  - **Azure Event Grid Transport**: सर्वरलेस इवेंट-ड्रिवन ट्रांसपोर्ट कार्यान्वयन
    - Azure Functions के साथ C#, TypeScript, और Python उदाहरण
    - स्केलेबल MCP समाधान के लिए इवेंट-ड्रिवन आर्किटेक्चर पैटर्न
    - वेबहुक रिसीवर्स और पुश-आधारित संदेश हैंडलिंग
  - **Azure Event Hubs Transport**: उच्च-थ्रूपुट स्ट्रीमिंग ट्रांसपोर्ट कार्यान्वयन
    - कम विलंबता परिदृश्यों के लिए रीयल-टाइम स्ट्रीमिंग क्षमताएं
    - विभाजन रणनीतियां और चेकपॉइंट प्रबंधन
    - संदेश बैचिंग और प्रदर्शन अनुकूलन
  - **एंटरप्राइज़ इंटीग्रेशन पैटर्न**: उत्पादन-तैयार आर्किटेक्चरल उदाहरण
    - कई Azure Functions में वितरित MCP प्रोसेसिंग
    - कई ट्रांसपोर्ट प्रकारों को मिलाकर हाइब्रिड ट्रांसपोर्ट आर्किटेक्चर
    - संदेश स्थायित्व, विश्वसनीयता, और त्रुटि हैंडलिंग रणनीतियां
  - **सुरक्षा और निगरानी**: Azure Key Vault इंटीग्रेशन और ऑब्ज़र्वेबिलिटी पैटर्न
    - प्रबंधित पहचान प्रमाणीकरण और न्यूनतम विशेषाधिकार पहुंच
    - Application Insights टेलीमेट्री और प्रदर्शन निगरानी
    - सर्किट ब्रेकर्स और फॉल्ट टॉलरेंस पैटर्न
  - **परीक्षण फ्रेमवर्क**: कस्टम ट्रांसपोर्ट्स के लिए व्यापक परीक्षण रणनीतियां
    - टेस्ट डबल्स और मॉकिंग फ्रेमवर्क के साथ यूनिट परीक्षण
    - Azure Test Containers के साथ इंटीग्रेशन परीक्षण
    - प्रदर्शन और लोड परीक्षण विचार

#### कॉन्टेक्स्ट इंजीनियरिंग (05-AdvancedTopics/mcp-contextengineering/) - उभरता हुआ AI अनुशासन
- **README.md**: कॉन्टेक्स्ट इंजीनियरिंग के रूप में एक उभरते हुए क्षेत्र की व्यापक खोज
  - **मुख्य सिद्धांत**: पूर्ण कॉन्टेक्स्ट साझा करना, क्रिया निर्णय जागरूकता, और कॉन्टेक्स्ट विंडो प्रबंधन
  - **MCP प्रोटोकॉल संरेखण**: MCP डिज़ाइन कॉन्टेक्स्ट इंजीनियरिंग चुनौतियों को कैसे संबोधित करता है
    - कॉन्टेक्स्ट विंडो सीमाएं और प्रगतिशील लोडिंग रणनीतियां
    - प्रासंगिकता निर्धारण और गतिशील कॉन्टेक्स्ट पुनर्प्राप्ति
    - मल्टी-मोडल कॉन्टेक्स्ट हैंडलिंग और सुरक्षा विचार
  - **कार्यान्वयन दृष्टिकोण**: सिंगल-थ्रेडेड बनाम मल्टी-एजेंट आर्किटेक्चर
    - कॉन्टेक्स्ट चंकिंग और प्राथमिकता तकनीक
    - प्रगतिशील कॉन्टेक्स्ट लोडिंग और संपीड़न रणनीतियां
    - स्तरित कॉन्टेक्स्ट दृष्टिकोण और पुनर्प्राप्ति अनुकूलन
  - **मापन फ्रेमवर्क**: कॉन्टेक्स्ट प्रभावशीलता मूल्यांकन के लिए उभरते मेट्रिक्स
    - इनपुट दक्षता, प्रदर्शन, गुणवत्ता, और उपयोगकर्ता अनुभव विचार
    - कॉन्टेक्स्ट अनुकूलन के लिए प्रयोगात्मक दृष्टिकोण
    - विफलता विश्लेषण और सुधार पद्धतियां

## 18 अगस्त, 2025

### दस्तावेज़ व्यापक अपडेट - MCP 2025-06-18 मानक

#### MCP सुरक्षा सर्वोत्तम प्रथाएं (02-Security/) - पूर्ण आधुनिकीकरण
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: आधिकारिक विनिर्देश के साथ संरेखित पूर्ण पुनर्लेखन
  - **अनिवार्य आवश्यकताएं**: आधिकारिक विनिर्देश से स्पष्ट MUST/MUST NOT आवश्यकताएं जोड़ी गईं
  - **12 मुख्य सुरक्षा प्रथाएं**: 15-आइटम सूची से व्यापक सुरक्षा डोमेन में पुनर्गठित
    - टोकन सुरक्षा और प्रमाणीकरण बाहरी पहचान प्रदाता इंटीग्रेशन के साथ
    - सत्र प्रबंधन और ट्रांसपोर्ट सुरक्षा क्रिप्टोग्राफिक आवश्यकताओं के साथ
    - AI-विशिष्ट खतरा सुरक्षा Microsoft Prompt Shields इंटीग्रेशन के साथ
    - एक्सेस कंट्रोल और अनुमतियां न्यूनतम विशेषाधिकार सिद्धांत के साथ
    - सामग्री सुरक्षा और निगरानी Azure Content Safety इंटीग्रेशन के साथ
    - आपूर्ति श्रृंखला सुरक्षा व्यापक घटक सत्यापन के साथ
    - OAuth सुरक्षा और Confused Deputy रोकथाम PKCE कार्यान्वयन के साथ
    - घटना प्रतिक्रिया और पुनर्प्राप्ति स्वचालित क्षमताओं के साथ
    - अनुपालन और शासन नियामक संरेखण के साथ
    - उन्नत सुरक्षा नियंत्रण शून्य विश्वास आर्किटेक्चर के साथ
    - Microsoft सुरक्षा पारिस्थितिकी तंत्र इंटीग्रेशन व्यापक समाधान के साथ
    - निरंतर सुरक्षा विकास अनुकूली प्रथाओं के साथ
  - **Microsoft सुरक्षा समाधान**: Prompt Shields, Azure Content Safety, Entra ID, और GitHub Advanced Security के लिए उन्नत इंटीग्रेशन गाइडेंस
  - **कार्यान्वयन संसाधन**: आधिकारिक MCP दस्तावेज़, Microsoft सुरक्षा समाधान, सुरक्षा मानक, और कार्यान्वयन गाइड द्वारा वर्गीकृत व्यापक संसाधन लिंक

#### उन्नत सुरक्षा नियंत्रण (02-Security/) - एंटरप्राइज़ कार्यान्वयन
- **MCP-SECURITY-CONTROLS-2025.md**: एंटरप्राइज़-ग्रेड सुरक्षा फ्रेमवर्क के साथ पूर्ण पुनर्लेखन
  - **9 व्यापक सुरक्षा डोमेन**: बुनियादी नियंत्रणों से विस्तारित एंटरप्राइज़ फ्रेमवर्क तक
    - उन्नत प्रमाणीकरण और प्राधिकरण Microsoft Entra ID इंटीग्रेशन के साथ
    - टोकन सुरक्षा और एंटी-पासथ्रू नियंत्रण व्यापक सत्यापन के साथ
    - सत्र सुरक्षा नियंत्रण अपहरण रोकथाम के साथ
    - AI-विशिष्ट सुरक्षा नियंत्रण प्रॉम्प्ट इंजेक्शन और टूल पॉइज़निंग रोकथाम के साथ
    - Confused Deputy हमले की रोकथाम OAuth प्रॉक्सी सुरक्षा के साथ
    - टूल निष्पादन सुरक्षा सैंडबॉक्सिंग और आइसोलेशन के साथ
    - आपूर्ति श्रृंखला सुरक्षा नियंत्रण निर्भरता सत्यापन के साथ
    - निगरानी और पहचान नियंत्रण SIEM इंटीग्रेशन के साथ
    - घटना प्रतिक्रिया और पुनर्प्राप्ति स्वचालित क्षमताओं के साथ
  - **कार्यान्वयन उदाहरण**: विस्तृत YAML कॉन्फ़िगरेशन ब्लॉक और कोड उदाहरण जोड़े गए
  - **Microsoft समाधान इंटीग्रेशन**: Azure सुरक्षा सेवाओं, GitHub Advanced Security, और एंटरप्राइज़ पहचान प्रबंधन का व्यापक कवरेज

#### उन्नत विषय सुरक्षा (05-AdvancedTopics/mcp-security/) - उत्पादन-तैयार कार्यान्वयन
- **README.md**: एंटरप्राइज़ सुरक्षा कार्यान्वयन के लिए पूर्ण पुनर्लेखन
  - **वर्तमान विनिर्देश संरेखण**: MCP विनिर्देश 2025-06-18 के साथ अपडेटेड अनिवार्य सुरक्षा आवश्यकताएं
  - **उन्नत प्रमाणीकरण**: Microsoft Entra ID इंटीग्रेशन के साथ व्यापक .NET और Java Spring Security उदाहरण
  - **AI सुरक्षा इंटीग्रेशन**: Microsoft Prompt Shields और Azure Content Safety कार्यान्वयन विस्तृत Python उदाहरणों के साथ
  - **उन्नत खतरा शमन**: व्यापक कार्यान्वयन उदाहरण
    - Confused Deputy हमले की रोकथाम PKCE और उपयोगकर्ता सहमति सत्यापन के साथ
    - टोकन पासथ्रू रोकथाम ऑडियंस सत्यापन और सुरक्षित टोकन प्रबंधन के साथ
    - सत्र अपहरण रोकथाम क्रिप्टोग्राफिक बाइंडिंग और व्यवहार विश्लेषण के साथ
  - **एंटरप्राइज़ सुरक्षा इंटीग्रेशन**: Azure Application Insights निगरानी, खतरा पहचान पाइपलाइंस, और आपूर्ति श्रृंखला सुरक्षा
  - **कार्यान्वयन चेकलिस्ट**: स्पष्ट अनिवार्य बनाम अनुशंसित सुरक्षा नियंत्रण Microsoft सुरक्षा पारिस्थितिकी तंत्र लाभों के साथ

- सभी विषयों को सटीक रूप से दर्शाने के लिए विज़ुअल पाठ्यक्रम मानचित्र को अपडेट किया गया
- उन्नत विषयों के विवरण को सभी विशेष क्षेत्रों को कवर करने के लिए बढ़ाया गया
- वास्तविक उदाहरणों को दर्शाने के लिए केस स्टडी अनुभाग को अपडेट किया गया
- यह व्यापक चेंजलॉग जोड़ा गया

### सामुदायिक योगदान (06-CommunityContributions/)
- छवि निर्माण के लिए MCP सर्वर्स के बारे में विस्तृत जानकारी जोड़ी गई
- VSCode में Claude का उपयोग करने पर व्यापक अनुभाग जोड़ा गया
- Cline टर्मिनल क्लाइंट सेटअप और उपयोग निर्देश जोड़े गए
- MCP क्लाइंट अनुभाग को सभी लोकप्रिय क्लाइंट विकल्पों को शामिल करने के लिए अपडेट किया गया
- अधिक सटीक कोड नमूनों के साथ योगदान उदाहरणों को बढ़ाया गया

### उन्नत विषय (05-AdvancedTopics/)
- सभी विशेष विषय फ़ोल्डरों को सुसंगत नामकरण के साथ व्यवस्थित किया गया
- कॉन्टेक्स्ट इंजीनियरिंग सामग्री और उदाहरण जोड़े गए
- Foundry एजेंट इंटीग्रेशन दस्तावेज़ीकरण जोड़ा गया
- Entra ID सुरक्षा इंटीग्रेशन दस्तावेज़ीकरण बढ़ाया गया
